# 🔐 Hash Generator Tool

A simple cybersecurity project that generates a **SHA-256 hash** from user input using **HTML, CSS, and JavaScript**.

## 📌 Overview
This tool allows a user to enter text and instantly generate a cryptographic hash. It demonstrates basic concepts in:

- Hashing
- Data integrity
- Browser-based cryptography
- Frontend security tooling

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- Web Crypto API

## 🚀 Features
- User input field
- Generate Hash button
- SHA-256 hash output
- Clean simple interface

## 💻 How to Use
1. Open `index.html` in your browser
2. Enter text into the input box
3. Click **Generate Hash**
4. View the SHA-256 hash output
